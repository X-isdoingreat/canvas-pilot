Canvas Pilot public preview

The first public preview of Canvas Pilot focuses on one-command workflows for recurring Canvas coursework.

What is included

  • Scan Canvas assignments into an approval plan.
  • Stop before execution until the student approves selected work.
  • Reuse course-specific recurring assignment patterns.
  • Produce review-ready drafts and per-assignment result files.
  • Write REPORT.md as the student-facing run closeout.
  • Keep private course data, runs, drafts, and credentials local.

Positioning

This release is not trying to be the largest Canvas API wrapper. It is the workflow layer above Canvas access. The public repo shows the framework for turning repeated course shapes into local, review-first workflows.

User boundary

Canvas Pilot is strongest for AI power users who already know how to use Codex or Claude Code. If you are comfortable running local agent workflows and reviewing outputs, it can remove a large amount of repeated coordination work. If not, it will feel like a developer tool.
