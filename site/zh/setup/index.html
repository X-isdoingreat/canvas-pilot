首次运行

严格按照这四步进行。

你只需要回答一个学校问题，再完成一次浏览器登录。本地安装和第一个 Skill 的分析都由 Agent 处理。

  1. 01

    下载

    本地 Agent 克隆 Canvas Pilot；如果已经下载，则保留现有配置并谨慎更新。

  2. 02

    告诉它学校

    Agent 先问你在哪所学校使用 Canvas，再定位官方登录页面；只有不明确时才追问网址。

  3. 03

    登录 Canvas

    Agent 打开交互式浏览器。SSO 和 2FA 只在学校 Canvas 页面输入。

  4. 04

    运行 Opportunity

    Agent 比较重复作业，推荐最适合的第一个 Skill，然后停下来等你选择编号。

安装之前，先判断作业是否适合。

Canvas Pilot 最适合真实要求和必要材料都能取得的作业。遇到缺口时，它应该明确报告，而不是假装完成。

完成课程配置后，尤其适合

代码作业

所有能在 Canvas 交付的代码题，哪怕没有正式 regression harness，也应优先考虑。Agent 仍要找到真实 spec、检查硬性限制并运行可用的复核。

定量与结构化商科作业

数学、会计、经济、统计、金融，以及结构化的 Word/Excel/PDF 商科作业，只要说明和输入可取得，都是强候选。

以客观题为主的 Canvas Quiz

受支持的选择、匹配、判断和数字题适配良好。多次机会、重试前可见反馈、取最高分，会让纠错闭环尤其有价值。

独立短答和短批注

worksheet、短答案和阅读批注，只要每个回答单元都短且可独立检查，即使合计超过 200 词也可以是强候选。

有条件支持

长段连续文字

主要连续 prose 单元约 200 词以上，是不把它选作第一个 Skill 的强降级信号。判断的不是总字数：许多各 10-40 词的独立回答仍可适配良好。

可访问的 instructor site、starter repository 或已配置 API 仍可用于取得真实 spec 和参考资料。

需要人工接手

任意外部交互网站

不要期待它稳定完成需要单独登录、CAPTCHA、反自动化检查、只看视频或未知浏览器流程的第三方网站作业。

New Quizzes/LTI、直接在线文本输入、LockDown Browser、监考、身份验证、线下任务、本人签名和最终 GradeScope 上传都需要你处理。

一个简单判断： 判断真实任务、数字化交付路径和反馈闭环，而不是只看文件扩展名或总字数。如果必须在陌生网站界面里完成操作，请预留人工接手。

安装结束时，作业还没有开始。

这段提示词只负责配置本地工具并停止。只有你以后明确提出请求，课程流程才会开始。

  1. 安装

    完成本地安装和浏览器登录，判断重复作业并生成定性机会榜单，然后停下来等你选择。

  2. 选择

    你选中一个候选项后，Canvas Pilot 才会验证并搭建对应的课程 Skill；选号不构成提交授权。

  3. 扫描

    以后你说 scan canvas 时，系统写出计划，然后再次停止。

  4. 批准

    你明确选择哪些计划项目可以运行。没有批准的作业不会执行。

  5. 执行

    已批准的流程生成草稿、单项结果和报告。提交从来不是静默默认。

人工检查仍然重要。 生成的草稿可能不完整。submitted 只表示 Canvas 收到了一次尝试，不代表答案正确，也不代表已经得到成绩。

默认留在本机，限制也说清楚。

Canvas Pilot 运行在本地 Agent 所在的机器上。登录、课程信息、源文件、草稿和运行状态都保留在本机。

你需要准备

  • Codex 或类似的本地 Agent，并且它有 shell 和文件访问能力。
  • Git 和 Python 3.11 或更高版本。
  • 可以打开交互式 Chromium 窗口的桌面环境。
  • 正常的 Canvas 登录和 2FA 方式。
  • 以后批准的作业所需的真实材料。

安装不授权这些操作

  • 安装时读取或执行 pending 作业。
  • 上传或提交任何 Canvas 作业。
  • 公开课程信息或凭据。
  • 从本地仓库 commit、push 或 deploy。
不要把秘密发到聊天里。 密码和 2FA 只输入 Canvas 浏览器。保留的 session cookie 也属于敏感的本地凭据。

复制之前，先确认这几件事。

可以使用浏览器里的普通聊天机器人吗？

不可以。安装需要本地 Agent 具备 shell、文件、Git、Python 访问能力，并能打开交互式浏览器。

安装会直接开始做作业吗？

不会。提示词明确禁止在安装期间扫描、规划、执行、上传或提交作业。Agent 必须在验证安装后停止。

可以读取课程外部网站吗？

可以读取可访问的外部规格和参考资料，但不承诺操作任意第三方作业网站。

遇到不支持的任务会怎样？

流程会记录 error 或 skipped，说明缺少的能力或材料，并给出人工下一步。它不应该伪造成功。
