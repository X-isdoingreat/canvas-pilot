# sources/

Drop your own assignment input materials here — readings (PDF/DOCX), spec
attachments, prior drafts, rendering scripts you want to keep around.

Everything inside this directory is **gitignored** by default. This file
(`README.md`) is the only thing tracked, so the directory exists in a fresh
clone and skills know where to look. Your actual materials never leave your
machine.

The framework's per-course skills look in here when an assignment's spec
points to a local file the student is expected to have supplied.
