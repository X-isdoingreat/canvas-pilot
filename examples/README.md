# Public-Safe Examples

This directory contains fake, public-safe examples that show how Canvas Pilot
turns recurring course patterns into reusable workflows.

All course names, assignment names, IDs, dates, and URLs in these examples are
invented. They are not copied from a real student, school, instructor, or
Canvas course.

`proposed_skill` values in these examples use public-safe demo labels. A real
installation may use different skill directory names.

Read the examples as one workflow:

1. Course-pattern files show what Canvas Pilot remembers across weeks.
2. `demo-plan.json` shows the scan artifact: every item starts unapproved.
3. `demo-transcript.md` shows the student approval handoff.
4. `demo-report.md` shows the execute closeout: approved items become
   `draft_ready`, and unapproved items become `skipped` audit records.
